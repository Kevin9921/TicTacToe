Tic Tac Toe

Developed Tic Tac Toe software utilizing Java language to create a basic interactive game requiring 2 players
Compiled a virtual Tic Tac Toe board where users can enter the row and column in which they like to place a tile
Integrated GUI which inherits code to build upon the logic by adding a user interface, importing Swing library
Interactive and able to handle a draw game scenario, with a total of 9 individual moves
